DEPLOYMENT INSTRUCTIONS
================================================================================

For GitHub Pages:

1. Create/Clone GitHub repository
2. Copy entire static-web-app folder to repository root or subdirectory
3. Update firebase-config.js with your Firebase credentials
4. Commit and push to GitHub
5. Go to repository Settings → Pages
6. Select branch and folder
7. Your site will be live at:
   https://username.github.io/repository-name/

For Local Development:

1. Use Python: python -m http.server 8000
2. Use Node.js: npx http-server
3. Use VS Code: Install Live Server extension
4. Navigate to http://localhost:8000
