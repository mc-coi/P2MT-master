Set Up Outlook Sending

Sending emails through Outlook requires a free one-time setup in Microsoft Azure (about 5 minutes).

  1. Go to portal.azure.com → App registrations and click New registration.
  2. Name the app anything (e.g. P2MT Email). Under Supported account types choose "Accounts in any organizational directory and personal Microsoft accounts". Under Redirect URI, select Single-page application (SPA) and enter your P2MT app URL (e.g. ).
  3. After registration, go to API permissions → Add a permission → Microsoft Graph → Delegated permissions. Search for and add Mail.Send, then click Grant admin consent.
  4. Copy the Application (client) ID from the app's Overview page and paste it below.
